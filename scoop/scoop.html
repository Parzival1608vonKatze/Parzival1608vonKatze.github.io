Scoop

Pass auf, dass PowerShell 5 (oder neuer, PowerShell Core enthalten) und .NET Framework 4.5 (or later)auf deinem PC installiert sind. Danach füre foldendes in Powershell aus:

       
           iwr -useb get.scoop.sh | iex

           # oder die einfacher verstehbarere langform

           Invoke-Expression (New-Object System.Net.WebClient).DownloadString('https://get.scoop.sh')
Notiz wenn Powershell einen fehler ausspuckt, probier forher den folgenden Befehl auszuführen: 
       
        Set-ExecutionPolicy RemoteSigned -scope CurrentUser
Sachen die man nach dem installieren machen kann 
       
        coop bucket add extras
        scoop bucket add nonportable
        scoop bucket add main
        scoop bucket add games
        
        #ein paar tools installieren
        scoop install python
        # aria2 für schnellere downloads 
        scoop install aria2
